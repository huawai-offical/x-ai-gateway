# X-22 关单收口（2026-04-13）

## 原始目标
- 实现候选路由打分、健康状态与 fallback 骨架。

## 实现结果
- `GatewayRouteSelectionService` 升级为候选规划器，输出：
  - `candidateEvaluations`
  - `attempts`
  - `healthState`
  - `scoreBreakdown`
  - `selectionSource`
- `GatewayChatExecutionService` 支持：
  - sync 首包前 fallback
  - stream 首个可见 chunk 前 fallback
  - 成功后重绑 affinity
  - 失败后写入 cooldown
- `GatewayResourceExecutionService` 支持：
  - JSON / binary / multipart 三类资源执行首包前 fallback
- `GatewayObservabilityService` 扩展候选摘要，记录评分、健康、attempt chain
- 新增仓库事实来源文档：
  - `docs/specs/routing-scoring-health-fallback-2026-04-13.md`
