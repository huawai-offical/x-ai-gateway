# X-23 关单收口（2026-04-13）

## 原始目标
- 接入 Redis 鉴权缓存、路由缓存与限流存储。

## 实现结果
- 新增 Redis 运行态抽象：
  - `AuthCacheStore`
  - `RouteCacheStore`
  - `RateLimitStore`
  - `HealthStateStore`
- 新增 Redis 实现：
  - `RedisAuthCacheStore`
  - `RedisRouteCacheStore`
  - `RedisRateLimitStore`
  - `RedisHealthStateStore`
- `DistributedKeyAuthenticationService` 接入 read-through 鉴权缓存
- `DistributedKeyGovernanceService` 下沉到 `RateLimitStore`
- `GatewayRouteSelectionService` 接入 `RouteCacheStore` 与 `HealthStateStore`
- 所有新 store 在 Redis 不可用时统一退化为 miss/no-op，避免本地与测试环境直接失败
- 新增仓库事实来源文档：
  - `docs/specs/redis-runtime-stores-2026-04-13.md`
