# x-ai-gateway

`x-ai-gateway` 是一个多协议、多厂商的 AI 网关，基于 `Spring Boot`、`Spring AI`、`PostgreSQL`、`Redis` 和 `React` 控制台实现，统一承载 OpenAI / Anthropic / Gemini 等协议入口、路由与能力判定、缓存与亲和、usage / audit / observability，以及后台管理能力。

项目当前默认数据库名为 `x_ai_gateway`。
